# Content Standards

## Non-Negotiables

1. No fabricated numbers, screenshots, benchmarks, or customer evidence.
2. No generic "AI can change everything" copy without a concrete angle.
3. No autoposting without explicit human approval.
4. No link-first posts unless there is a deliberate tradeoff and the user accepts the reach hit.

## Quality Bar

Every publish-ready draft should be:

- specific about who it is for
- grounded in real observation, data, or direct experience
- easy to scan in the first two lines
- clear about the one takeaway
- aligned with the brand voice

## Hook Standard

Hooks should usually include at least two of the following:

- tension
- specificity
- credible experience
- a concrete outcome

Hooks that are merely vague curiosity bait should be rejected.

## Voice Standard

The default voice is:

- direct
- intelligent
- practical
- lightly opinionated

Avoid:

- faux inspiration
- filler adjectives
- obvious AI phrasing
- empty controversy

## Review Standard

If a draft is technically strong but strategically weak, revise it.

If a draft is strategically strong but factually weak, block it.

Accuracy outranks style.
